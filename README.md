7. Key Insights
Customers on month-to-month contracts have 3x higher churn
High monthly charges strongly correlate with churn
Most churn occurs within first 60 days

8. Recommendations
Offer discounts for long-term contracts
Improve onboarding experience
Target high-risk users early
